# RSS Fetcher Skill

This skill helps you fetch and parse RSS/Atom feeds from any URL.

## Usage

1. Provide an RSS feed URL
2. The skill will fetch and parse the feed
3. Return article titles, links, and publication dates

## Examples

- Fetch latest news from Hacker News
- Get updates from tech blogs
- Monitor industry news

## Notes

- Supports RSS 2.0 and Atom formats
- Handles common feed structures
- Can filter by date range
