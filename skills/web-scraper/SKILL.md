# Web Scraper Skill

This skill helps you extract content from web pages.

## Usage

1. Provide a URL to scrape
2. The skill fetches the page
3. Extracts relevant content
4. Returns structured data

## Examples

- Extract article text from news sites
- Get product information from e-commerce
- Parse job listings
- Extract contact information

## Notes

- Handles various HTML structures
- Can bypass basic Cloudflare
- Returns clean text content
- Respects robots.txt where applicable
