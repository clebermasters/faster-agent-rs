# Code Review Skill

This skill helps you review code for issues and improvements.

## Usage

1. Provide code to review
2. Specify language and context
3. Get comprehensive analysis
4. Review findings and suggestions

## Areas Analyzed

- Security vulnerabilities
- Performance issues
- Code smells
- Best practices violations
- Error handling
- Resource management
- Test coverage

## Output Format

Each finding includes:
- Severity (Critical/High/Medium/Low)
- Location (file:line)
- Description
- Suggestion for fix
