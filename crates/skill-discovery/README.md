# skill-discovery

Semantic and keyword skill search engine for the [Skill Agent](https://github.com/clebermasters/faster-agent-rs) framework.

Combines vector similarity search (via `skill-embeddings`) with keyword/trigger matching for hybrid skill discovery.

## Part of the Skill Agent workspace

See the [main repository](https://github.com/clebermasters/faster-agent-rs) for full documentation.
