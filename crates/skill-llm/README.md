# skill-llm

LLM client integrations and agent execution loops for the [Skill Agent](https://github.com/clebermasters/faster-agent-rs) framework.

Supports MiniMax, Ollama, and AWS Bedrock. Provides `Agent` and `StreamingAgent` with tool-calling loops.

## Part of the Skill Agent workspace

See the [main repository](https://github.com/clebermasters/faster-agent-rs) for full documentation.
