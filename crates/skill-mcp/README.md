# skill-mcp

Model Context Protocol (MCP) integration for the [Skill Agent](https://github.com/clebermasters/faster-agent-rs) framework.

Connects to MCP servers defined in `mcp.json` and exposes their tools to the agent loop.

## Part of the Skill Agent workspace

See the [main repository](https://github.com/clebermasters/faster-agent-rs) for full documentation.
